STEP 2 — COPY THESE 3 FILES TO YOUR PROJECT
=============================================
Add all three files to the ROOT of your GitHub repo.

  AIRULES.md      <- fill in your project name and stack
  CONTEXT.md      <- fill in your project name and stack
  PROMPT-LOG.md   <- leave empty, V0 fills it over time

How to add them:
  GitHub web: go to your repo -> Add file -> Upload files
  Terminal:   git add . && git commit -m "feat: add Agile99 core files" && git push

After adding them, V0 can read them at the start of every chat.
